# KeplerX

## NASA Space Apps Challenge 2025: Hunting for Exoplanets

KeplerX es una plataforma integral diseñada para la detección y análisis de exoplanetas, desarrollada como solución para el desafío "A World Away" del NASA Space Apps Challenge. El sistema utiliza técnicas de Machine Learning para procesar datos astronómicos masivos y ofrecer predicciones precisas a través de una interfaz web intuitiva.

## Descripción del Proyecto

Este proyecto fue creado para facilitar la identificación de potenciales exoplanetas a partir de datasets de la NASA. El núcleo del sistema es un modelo de Gradient Boosting entrenado para clasificar señales astronómicas, respaldado por una arquitectura full-stack que permite a los usuarios visualizar los resultados de manera efectiva.

## Tecnologías Principales

Backend: FastAPI (Python)

Frontend: HTML5, CSS3, Vanilla JavaScript

Machine Learning: Scikit-Learn, Pandas, Jupyter

Datos: NASA Open Datasets
